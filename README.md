# AWS Resource Usage Report using AWS CLI & Cron

## Overview

This project demonstrates how to automate AWS resource reporting using the AWS Command Line Interface (AWS CLI), Bash scripting, and Cron on Linux.

The script retrieves information about AWS resources and generates a simple usage report. The report can also be scheduled to run automatically using Cron, making it useful for basic cloud monitoring and automation.

---

## Objectives

* Install and configure AWS CLI on Linux
* Create and configure an IAM user for AWS CLI access
* Write a Bash script to report AWS resource usage
* Automate script execution using Cron
* Practice Linux command-line automation
